2.1  หน้าหลักผู้พักอาศัย (/home)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  หลังจาก Login สำเร็จ จะเข้ามาที่หน้าหลักซึ่งแสดงเมนูหลักทั้งหมด:

  ┌────────────────────────────────────────────────┐
  │  🏢 DORM SYSTEM         👤 ชื่อผู้ใช้ (ห้อง)   │
  │────────────────────────────────────────────────│
  │                                                │
  │   💰 ใบแจ้งหนี้    🏢 จองพื้นที่    🔧 แจ้งซ่อม  │
  │                                                │
  │   📢 ประกาศ        📦 พัสดุ         🔒 เปลี่ยนรหัส│
  │                                                │
  └────────────────────────────────────────────────┘

  เมนูที่ใช้ได้:
  • 💰 ใบแจ้งหนี้         — ดูใบแจ้งหนี้ / ชำระเงิน
  • 🏢 จองพื้นที่ส่วนกลาง — จองห้องประชุม, สระว่ายน้ำ ฯลฯ
  • 🔧 แจ้งซ่อม          — แจ้งปัญหาในห้องพัก
  • 📢 ประกาศ            — อ่านข่าวสารจากหอพัก
  • 📦 พัสดุ              — ตรวจสอบพัสดุที่รอรับ
  • 🔒 เปลี่ยนรหัสผ่าน     — เปลี่ยนรหัสผ่านบัญชี

  ปุ่มด้านบน:
  • "เชื่อมต่อ LINE"  — เชื่อมบัญชี LINE เพื่อรับแจ้งเตือน
  • "ออกจากระบบ"    — Logout กลับหน้า Login

2.2  ใบแจ้งหนี้ (/invoice)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  2.2.1  การดูใบแจ้งหนี้
  ──────────────────────

  1. กดเมนู "💰 ใบแจ้งหนี้" ที่หน้าหลัก
  2. ระบบแสดงสถิติสรุป:
     - จำนวนใบแจ้งหนี้ทั้งหมด
     - ยอดเงินรวม
     - จ่ายแล้ว / รอตรวจสลิป / รอชำระ
  3. เลือกเดือน/ปี เพื่อกรองใบแจ้งหนี้ตามช่วงเวลา
  4. แต่ละใบแจ้งหนี้แสดงรายละเอียด:

     ┌──────────────────────────────────┐
     │  🧾 ใบแจ้งหนี้ เดือน 06/2025     │
     │  ห้อง: A101                      │
     │  สถานะ: [รอชำระเงิน]             │
     │──────────────────────────────────│
     │  ค่าน้ำ     : ฿200               │
     │    (10→20 หน่วย × ฿20/หน่วย)     │
     │  ค่าไฟฟ้า   : ฿300               │
     │    (100→160 หน่วย × ฿5/หน่วย)    │
     │  ค่าส่วนกลาง: ฿400               │
     │  ค่าเช่าห้อง : ฿3,500            │
     │  รายการเพิ่ม : ฿100 (ค่าอินเทอร์เน็ต)│
     │──────────────────────────────────│
     │  รวมทั้งสิ้น : ฿4,500            │
     └──────────────────────────────────┘

  สูตรคำนวณ:
    ค่าน้ำ   = (มิเตอร์ปัจจุบัน - มิเตอร์ก่อนหน้า) × ราคาต่อหน่วย
    ค่าไฟฟ้า = (มิเตอร์ปัจจุบัน - มิเตอร์ก่อนหน้า) × ราคาต่อหน่วย
    รวม     = ค่าน้ำ + ค่าไฟ + ค่าส่วนกลาง + ค่าเช่า + รายการเพิ่ม


  2.2.2  สถานะใบแจ้งหนี้
  ──────────────────────

  สถานะ            ความหมาย
  ────────────────  ──────────────────────────────
  🔴 รอชำระเงิน     ยังไม่ได้ชำระ ต้องอัปโหลดสลิป
  🟡 รอตรวจสลิป     อัปโหลดสลิปแล้ว รอ Admin ตรวจสอบ
  🟢 จ่ายแล้ว       Admin อนุมัติการชำระเรียบร้อย


  2.2.3  การอัปโหลดสลิปชำระเงิน
  ────────────────────────────

  1. ที่ใบแจ้งหนี้ที่สถานะ "รอชำระเงิน" จะเห็น QR Code
  2. สแกน QR Code เพื่อชำระเงินผ่านธนาคาร/พร้อมเพย์
  3. กดปุ่ม "📤 อัปโหลดสลิป"
  4. เลือกไฟล์ภาพสลิปจากเครื่อง (รองรับ JPG, PNG)
  5. ระบบจะอัปโหลดสลิปและเปลี่ยนสถานะเป็น "รอตรวจสลิป"
  6. รอ Admin ตรวจสอบและอนุมัติ

  ⚠️ หมายเหตุ: อัปโหลดสลิปได้เฉพาะใบแจ้งหนี้ที่สถานะ "รอชำระเงิน"


  2.2.4  การดาวน์โหลดใบแจ้งหนี้ PDF
  ─────────────────────────────────

  1. ที่ใบแจ้งหนี้ กดปุ่ม "📄 ดาวน์โหลด PDF"
  2. ระบบจะสร้างไฟล์ PDF และดาวน์โหลดลงเครื่อง
  3. ไฟล์ PDF จะมีรายละเอียดครบทุกรายการ
  4. สามารถพิมพ์หรือส่งต่อได้

2.3  แจ้งซ่อม (/repair)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  2.3.1  การสร้างคำขอแจ้งซ่อม
  ──────────────────────────

  1. กดเมนู "🔧 แจ้งซ่อม" ที่หน้าหลัก
  2. กรอกข้อมูล:
     - หัวข้อ (Title): เช่น "หลอดไฟห้องนอนเสีย"
     - รายละเอียด (Description): อธิบายปัญหาให้ชัดเจน
       เช่น "หลอดไฟ LED ในห้องนอนกระพริบและดับ ต้องการเปลี่ยน"
  3. กดปุ่ม "📩 ส่งคำขอซ่อม"
  4. ระบบสร้างคำขอและตั้งสถานะเป็น "รอดำเนินการ"


  2.3.2  สถานะคำขอซ่อม
  ─────────────────────

  สถานะ              ความหมาย
  ──────────────────  ──────────────────────────
  🔴 รอดำเนินการ      ส่งคำขอแล้ว รอ Admin ดำเนินการ
  🟡 กำลังดำเนินการ    Admin รับเรื่องแล้ว อยู่ระหว่างซ่อม
  🟢 เสร็จสิ้น         ดำเนินการซ่อมเสร็จเรียบร้อย


  2.3.3  การติดตามและยกเลิก
  ────────────────────────

  • ดูรายการคำขอทั้งหมดของตัวเองในหน้าเดียวกัน
  • สามารถค้นหาด้วยคำสำคัญหรือเลขห้อง
  • ดูสถิติ: ทั้งหมด / รอดำเนินการ / กำลังดำเนินการ / เสร็จสิ้น
  • กดปุ่ม "ยกเลิก" ได้ เฉพาะคำขอที่สถานะ "รอดำเนินการ" เท่านั้น


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2.4  จองพื้นที่ส่วนกลาง (/facility-booking)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  2.4.1  การสร้างคำขอจอง
  ──────────────────────

  1. กดเมนู "🏢 จองพื้นที่ส่วนกลาง"
  2. กรอกข้อมูลการจอง:
     - ชื่อสถานที่/กิจกรรม: เช่น "ห้องประชุม", "สระว่ายน้ำ"
     - วันที่จอง: เลือกจากปฏิทิน
     - เวลาเริ่ม: เช่น 10:00
     - เวลาสิ้นสุด: เช่น 12:00
     - จำนวนผู้ใช้: ระบุจำนวนคน (1-50 คน)
  3. กดปุ่ม "📩 ส่งคำขอจอง"
  4. สถานะเริ่มต้นเป็น "รอยืนยัน"


  2.4.2  สถานะการจอง
  ──────────────────

  สถานะ            ความหมาย
  ────────────────  ──────────────────────────────
  🟡 รอยืนยัน       ส่งคำขอแล้ว รอ Admin อนุมัติ
  🟠 กำลังจะถึง     Admin อนุมัติแล้ว ยังไม่ถึงเวลา
  🟢 เสร็จสิ้น       ใช้งานเสร็จแล้ว (เลยเวลาสิ้นสุด)
  🔴 ถูกปฏิเสธ      Admin ปฏิเสธคำขอจอง

  หมายเหตุ: สถานะ "กำลังจะถึง" และ "เสร็จสิ้น" คำนวณอัตโนมัติ
  จากเวลาจริงเทียบกับเวลาจอง


  2.4.3  การยกเลิกการจอง
  ──────────────────────

  • สามารถยกเลิกได้หากเวลาปัจจุบัน < 1 ชั่วโมง ก่อนเวลาจอง
  • กดปุ่ม "ยกเลิก" ที่รายการจองที่ต้องการ
  • หลังจาก 1 ชั่วโมงก่อนเวลาจอง จะไม่สามารถยกเลิกได้

  ตัวอย่าง: จองเวลา 14:00
  → ยกเลิกได้ถึง 12:59 (ก่อน 1 ชม.)
  → 13:00 เป็นต้นไป ไม่สามารถยกเลิกได้

2.5  ประกาศข่าวสาร (/announcements)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. กดเมนู "📢 ประกาศ"
  2. ระบบแสดงรายการประกาศทั้งหมดจากผู้ดูแลหอพัก
  3. แต่ละประกาศแสดง:
     - วันที่โพสต์
     - ชื่อผู้โพสต์
     - หัวข้อประกาศ
     - เนื้อหา
  4. เรียงลำดับจากล่าสุดอยู่บนสุด
  5. แสดงจำนวนประกาศทั้งหมด

  หมายเหตุ: ผู้พักอาศัยสามารถอ่านได้อย่างเดียว ไม่สามารถสร้าง/แก้ไข

2.6  พัสดุ (/parcel)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  2.6.1  การตรวจสอบพัสดุ
  ──────────────────────

  1. กดเมนู "📦 พัสดุ"
  2. ระบบแสดงสถิติ:
     - จำนวนพัสดุทั้งหมด
     - พัสดุใหม่ (มาใน 24 ชม. ยังไม่ได้รับ)
     - รับแล้ว
  3. รายการพัสดุแสดง:
     - รายละเอียดพัสดุ
     - วันเวลาที่รับเข้าระบบ
     - ชื่อผู้ส่ง / เลขพัสดุ (หากมี)
     - สถานะ: "ใหม่" หรือ "รับแล้ว"
     - จำนวนวันที่รอรับ

  4. สามารถค้นหาด้วยรายละเอียด, ชื่อผู้ส่ง หรือเลขพัสดุ


  2.6.2  การยืนยันรับพัสดุ
  ────────────────────────

  1. ที่รายการพัสดุที่สถานะ "ใหม่"
  2. กดปุ่ม "ยืนยันรับพัสดุ"
  3. ระบบบันทึกว่ารับแล้ว พร้อมวันเวลาที่รับ


  2.6.3  การแจ้งเตือนผ่าน LINE
  ────────────────────────────

  • เมื่อ Admin ลงทะเบียนพัสดุใหม่ ระบบจะส่ง LINE แจ้งเตือนอัตโนมัติ
  • ข้อความจะถูกส่งเป็น Flex Message ไปยัง LINE ของผู้พักอาศัย
  • ต้องเชื่อมบัญชี LINE กับระบบก่อนจึงจะรับแจ้งเตือนได้
    (กดปุ่ม "เชื่อมต่อ LINE" ที่หน้าหลัก)

2.7  เปลี่ยนรหัสผ่าน (/change-password)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  2.7.1  ขั้นตอนการเปลี่ยนรหัสผ่าน
  ─────────────────────────────────

  1. กดเมนู "🔒 เปลี่ยนรหัสผ่าน"
  2. กรอกข้อมูล:

     ┌──────────────────────────────────────┐
     │  🔒 รหัสผ่านเดิม:     [___________] 👁️│
     │  🔑 รหัสผ่านใหม่:     [___________] 👁️│
     │  ✅ ยืนยันรหัสผ่าน:    [___________] 👁️│
     │                                      │
     │  [ บันทึกรหัสผ่านใหม่ ]               │
     └──────────────────────────────────────┘

  3. กดไอคอน 👁️ เพื่อแสดง/ซ่อนรหัสผ่าน
  4. กดปุ่ม "บันทึกรหัสผ่านใหม่"
  5. ถ้าสำเร็จ ระบบจะแสดง "เปลี่ยนรหัสผ่านสำเร็จ!"
     และ Logout อัตโนมัติใน 2 วินาที
  6. เข้าสู่ระบบใหม่ด้วยรหัสผ่านใหม่


  2.7.2  เงื่อนไขรหัสผ่าน
  ────────────────────────

  ✅ ต้องกรอกครบทั้ง 3 ช่อง
  ✅ รหัสผ่านใหม่ต้องมีอย่างน้อย 6 ตัวอักษร
  ✅ รหัสผ่านใหม่ต้องตรงกับช่องยืนยัน
  ✅ รหัสผ่านใหม่ต้องไม่เหมือนรหัสเดิม
  ✅ รหัสผ่านเดิมต้องถูกต้อง

  • ใช้ตัวอักษรอย่างน้อย 6 ตัว
