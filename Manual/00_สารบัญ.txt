  ภาพรวมระบบ
═══════════════════════════════════════

ระบบนี้เป็น Web Application สำหรับบริหารจัดการหอพัก รองรับ 2 บทบาท:

  • ผู้พักอาศัย (Resident) — ดูใบแจ้งหนี้, แจ้งซ่อม, จองพื้นที่,
    ชำระเงิน, ดูประกาศ, ตรวจสอบพัสดุ

  • ผู้ดูแลระบบ (Admin) — สร้างใบแจ้งหนี้, อนุมัติคำขอ,
    จัดการผู้ใช้, ลงทะเบียนพัสดุ, ส่งแจ้งเตือน LINE

ฟีเจอร์หลัก 7 ระบบ:
  1. ระบบจัดการใบแจ้งหนี้ (Invoice)
  2. ระบบแจ้งซ่อม (Repair Request)
  3. ระบบจองพื้นที่ส่วนกลาง (Facility Booking)
  4. ระบบจัดการพัสดุ + LINE Notification (Parcel)
  5. ระบบประกาศข่าวสาร (Announcement)
  6. ระบบจัดการผู้ใช้ (User Management)
  7. ระบบ Admin Logs (Audit Trail)

เทคโนโลยี:
  Frontend   — Angular 20 + TypeScript
  Backend    — Node.js + Express 5
  Database   — MySQL 8.0
  Auth       — JWT + bcrypt
  Notify     — LINE Messaging API

URL เข้าใช้งาน:
  http://localhost:4200
