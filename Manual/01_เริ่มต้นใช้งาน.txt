1.1  ความต้องการขั้นต่ำ
═══════════════════════

  • เว็บเบราว์เซอร์ที่รองรับ: Google Chrome, Microsoft Edge, Firefox
    (แนะนำ Chrome เวอร์ชันล่าสุด)
  • การเชื่อมต่ออินเทอร์เน็ต (สำหรับ LINE Notification)
  • บัญชี LINE (หากต้องการรับแจ้งเตือนพัสดุ)


1.2  การเข้าสู่ระบบ (Login)
════════════════════════════

  1. เปิดเว็บเบราว์เซอร์ เข้าไปที่ URL:
     http://localhost:4200

  2. ระบบจะแสดงหน้า Login ให้กรอกข้อมูล:

     ┌──────────────────────────────────────┐
     │  🏢 FACILITY AND SERVICE SYSTEM      │
     │                                      │
     │  👤 Username:  [_______________]     │
     │  🔒 Password:  [_______________]     │
     │                                      │
     │  ☐ จดจำฉัน (Remember Me)             │
     │                                      │
     │  [ เข้าสู่ระบบ ]                      │
     │  [ เข้าสู่ระบบด้วย LINE ]             │
     └──────────────────────────────────────┘

  3. กรอก Username และ Password ที่ได้รับจากผู้ดูแลหอพัก

  4. กดปุ่ม "เข้าสู่ระบบ"

  5. หากข้อมูลถูกต้อง ระบบจะนำไปยังหน้าหลัก:
     - ผู้พักอาศัย → หน้า /home
     - ผู้ดูแลระบบ  → หน้า /staff

  6. หากข้อมูลไม่ถูกต้อง จะแสดงข้อความ Error:
     - "User not found"        — ไม่พบชื่อผู้ใช้
     - "Invalid credentials"   — รหัสผ่านไม่ถูกต้อง


1.3  Remember Me (จดจำฉัน)
═══════════════════════════

  • ทำเครื่องหมาย ☑ ที่ "จดจำฉัน" ก่อนกด Login
  • ครั้งต่อไปที่เปิดหน้า Login ระบบจะกรอก Username ให้อัตโนมัติ
  • เหมาะสำหรับเครื่องคอมพิวเตอร์ส่วนตัว

  ⚠️ คำเตือน: ไม่ควรใช้ Remember Me บนคอมพิวเตอร์สาธารณะ


1.4  การเข้าสู่ระบบด้วย LINE
══════════════════════════════

  1. กดปุ่ม "เข้าสู่ระบบด้วย LINE"
  2. ระบบจะพาไปยังหน้า LINE Login (OAuth)
  3. ยืนยันตัวตนด้วยบัญชี LINE
  4. ระบบจะเชื่อมโยง LINE กับบัญชีผู้ใช้
  5. หลังจากเชื่อมต่อแล้ว จะรับแจ้งเตือนพัสดุผ่าน LINE ได้

  หมายเหตุ: ต้องมีบัญชีผู้ใช้ในระบบก่อนจึงจะเชื่อมต่อ LINE ได้


1.5  Session Timeout (หมดเวลา)
═══════════════════════════════

  ระบบมีการจำกัดเวลาการใช้งาน เพื่อความปลอดภัย:

  บทบาท         เวลาหมดอายุ     คำเตือนก่อนหมด
  ─────────────  ──────────────  ────────────────
  ผู้พักอาศัย     5 นาที          1 นาทีก่อน
  ผู้ดูแลระบบ     20 นาที         1 นาทีก่อน

  • หากไม่มีการใช้งาน (ไม่ขยับเม้าส์, ไม่พิมพ์) เกินเวลาที่กำหนด
    ระบบจะแสดงคำเตือน และ Logout อัตโนมัติ
  • การขยับเม้าส์, กดคีย์บอร์ด, หรือแตะหน้าจอ จะรีเซ็ตตัวนับเวลา


1.6  การออกจากระบบ (Logout)
════════════════════════════

  1. กดปุ่ม "ออกจากระบบ" ที่แถบเมนูด้านบน
  2. ระบบจะลบ Token และนำกลับไปหน้า Login

  หมายเหตุ: เมื่อเปลี่ยนรหัสผ่านสำเร็จ ระบบจะ Logout อัตโนมัติ


1.7  บทบาทผู้ใช้งาน
═════════════════════

  บทบาท           สิทธิ์การใช้งาน
  ───────────────  ──────────────────────────────────
  ผู้พักอาศัย      ดูใบแจ้งหนี้, อัปโหลดสลิป, แจ้งซ่อม,
  (resident)       จองพื้นที่, ดูประกาศ, ดูพัสดุ,
                   เปลี่ยนรหัสผ่าน

  ผู้ดูแลระบบ      ทุกอย่างของผู้พักอาศัย + สร้างใบแจ้งหนี้,
  (admin)          อนุมัติคำขอ, จัดการผู้ใช้, ลงทะเบียนพัสดุ,
                   จัดการประกาศ, ดู Admin Logs
