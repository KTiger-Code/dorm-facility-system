📋 คำอธิบาย External Entities

  • ผู้พักอาศัย (Resident): ผู้ใช้ที่พักอาศัยในหอพัก สามารถดูใบแจ้งหนี้ แจ้งซ่อม จองพื้นที่ส่วนกลาง ดูประกาศ และรับแจ้งเตือนพัสดุ
  • ผู้ดูแลระบบ (Admin/Staff): เจ้าหน้าที่หอพัก มีสิทธิ์จัดการข้อมูลทั้งหมด รวมถึงการสร้างใบแจ้งหนี้ อนุมัติการจอง จัดการพัสดุ และดูแลผู้ใช้
  • LINE Platform: ระบบ LINE Messaging API ใช้สำหรับส่งแจ้งเตือนพัสดุถึงผู้พักอาศัยแบบอัตโนมัติผ่าน LINE Bot
  • MySQL Database: ฐานข้อมูล MySQL 8.0 (dorm_pwa) เก็บข้อมูลผู้ใช้ ใบแจ้งหนี้ คำขอซ่อม การจอง พัสดุ ประกาศ และ Admin Logs
